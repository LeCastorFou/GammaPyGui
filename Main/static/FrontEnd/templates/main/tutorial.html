The Gamma Py Gui Tutorial

This is an open source project allowing anyone to use this GUI to performe Gamma Py analysis. Comment, questions and improvement are welcome an can be notify in the GitHub

Config set up

First of all the user must defined a path to his data local storage. Go to "Account config". For the moment only H.E.S.S. data path are available. In the form you must provide the absolute path where are located the hdu-index.fits and the obs-index.fits. Once this is done the GUI should automaticaly propose to you the available sources for analysis.

H.E.S.S analysis

To perform H.E.S.S. Analysis , go to "Analysis" then on the left menu chosse "H.E.S.S. analysis". Belllow the left menu you have a form that you can fill.

  • Choose a name for your analysis (if you already performed an analysis with this name an index _%i is added)
  • Select an available source (depends on your fits files location see above)
  • Defined a min an a max run where to performe the analysis
  • Or select a run list in txt format
  • Clic on Launch analysis
  • Once the browser stops loading ,your analysis is performed. Your results file are available in $PATHTOGUI/GammaPyGui/Main/static/results/NAME_OF_ANALYSIS

H.E.S.S spectrum

  • Choose a name for your analysis (you can choose an existing analysis folder)
  • Select an available source (depends on your fits files location see above)
  • Defined a min an a max run where to performe the analysis
  • Or select a run list in txt format
  • Clic on Launch spectrum
  • Once the browser stops loading ,your analysis is performed. Your results file are available in $PATHTOGUI/GammaPyGui/Main/static/results/NAME_OF_ANALYSIS

CTA simulations

  • Choose a duration for the observation
  • Select the source index and flux at 1 Tev in cm-2 s-1 TeV-1 (ex:1e-11)
  • Select Pointing position and source position
  • Click on "Launch Simu" : after loading a plot should appear

Results files

If you go to the 'Results' menu you will have acces to all you locally saved results. Select one an get acces to the results files pictures.
